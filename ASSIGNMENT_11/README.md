# Assignment 11 - Library System (Abstraction & Polymorphism)

## Question
Design a library system in Java or Python.

### Base/Abstract Class: LibraryItem
- Common fields: title, year
- Abstract/common method: displayInfo()

### Subclasses:
- **Book**: add author
- **DVD**: add duration/genre

### Must Demonstrate:
1. Abstraction (common structure in base class)
2. Method overriding in subclasses
3. Polymorphism using a collection of LibraryItem objects
4. One additional feature: constructor overloading / default arguments OR class/static counter

## Language
Java or Python

## Files
- \LibrarySystem.java\ or \library_system.py\
