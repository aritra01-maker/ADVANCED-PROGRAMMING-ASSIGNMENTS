# Assignment 19 - Digital Counter & Theme Toggle App (React Native)

## Question
Build a single-screen mobile application using React Native.

### Features:
- Digital counter (increment, decrement, reset)
- Theme toggle (Light Mode / Dark Mode)

### Counter Logic:
- Starts at 0
- Increment: +1
- Decrement: -1 (never goes below 0)
- Reset: back to 0

### Theme:
- Light Mode (default): white background, dark text
- Dark Mode: dark gray/black background, white text
- Toggle Theme button swaps styles instantly

### Must Use:
- View, Text, TouchableOpacity (or Button)
- useState for counter value and theme mode
- Flexbox for layout (increment/decrement buttons side-by-side)
- Conditional styling using ternary operators
